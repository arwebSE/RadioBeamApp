Support

Help for station playback, CarPlay, favorites, and app setup.

Get Help

For RadioBeam support, report an issue in the AppleGarden GitHub repository:

Open a support issue

Useful Details to Include

  • Your iPhone model and iOS version.
  • Whether the issue happens in the iPhone app, CarPlay, or both.
  • The station name, country, and stream URL if a specific station is affected.
  • Whether playback starts, buffers, stops, or shows metadata incorrectly.

CarPlay Notes

If RadioBeam does not appear in CarPlay, confirm the app is installed on the connected iPhone and that CarPlay is enabled for the vehicle or head unit. Some simulator-only CarPlay behavior differs from real devices.

Station Availability

RadioBeam streams third-party radio stations. If a station is offline, geo-blocked, or has changed its stream URL, playback may fail until the station directory is updated.
